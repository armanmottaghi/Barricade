To run the simulation, follow the steps below:

1. Run Main.m file in MATLAB.
2. Modify the paramters listed under the "Initialize" section.
3. Cick on the Run button in Matlab.
4. Close the figure.
5. Repeat steps 1-4 if necessary.

Note: If you have any questions about the methodology or the project iteself, please refer to Fu, Li, Mottaghi - The Barricade .docx file.
